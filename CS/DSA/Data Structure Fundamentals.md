
Data structures are a way to structure data, specifically in RAM (Random Access Memory)
This is where variables are stored, in memory. RAM is contiguous.

RAM is stored in bytes.

1 byte = 8 bits.

An integer (number) is 4 bytes (32 bits)
A character is 1 byte (8 bits)

When storing an array, (a continuous string of letters or numbers etc.) it is stored continuously, and in order.
